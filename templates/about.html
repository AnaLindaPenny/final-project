Brief Project Summary

Our Inspiration:

As a team we choose a topic that was fun and exciting to explore in more detail, while still being informative. We are all MARVEL &/or DC fanatics so we have all thoroughly enjoyed analyzing the various data resources for this project. Our POW-er stricken project had some BANG!-type experiences but overall the challenges that we encountered, have continued to strenghten our capabilities so that we could get to the ENDGAME a.k.a the Heroku Cloud Application Platform deployment.

Data Resources Review & Analysis:

We reviewed a variety of data content from the MARVEL vs. DC IMDB dataset. The data content was overall quite extensive so we wrote various scripts to web scrape the data.

Extract, Load & Transform:

We collectively developed various scripts to web scrape only the relevant data from the diffent sections of the websites.

Languages & Libraries Utilized:

Jupyter Notebook; Python (Pandas, GeoPandas, Web-Scrapping, BeautifulSoup); Tableau; Machine Learning (Scikit-Learn); MongoDB; JavaScript (D3.js, Plotly.js, Chartist.js); HTML, CSS (Bootstrap); JSON (GeoJSON); Heroku.

Website Deployment:

We deployed via Heroku Cloud Application Platform.
