Box Office Predictions

Below you'll see the results of machine learning for the MCU and the DCEU.


MCU Heatmap


DCEU Heatmap


The first step was to create a heatmap to emphasize the correlation between all the variables using seaborn. As you can see, the darker the color, the higher the positivie correlation. The more yellow the color is, the closer it is to negative correlation.

Some interesting things to notice are how on the Marvel heatmap, the lowest correlation is between any variable and number of weeks in the theater. For the DCEU heatmap, the lowest correlation is between any variable and Harley Quinn or Shazam.

This image is condensed to emphasize the correlation by color, scroll down to see another seaborn correlation format.


MCU Pair Plot


DCEU Pair Plot


This is another generated image from seaborn called a pairplot. It does the same type of comparison as the heatmap, but instead of giving a color based on correlation, it shows what the graphs would look like for each comparison. The bar chart graphs it created are comparing the same variables.

This was very helpful in planning which graphs to create to do machine learning on.

MCU Sample Predictions


DCEU Sample Predictions


The first machine learning visualization we created takes a sample of the movie data and calculates its worldwide gross based on it's budget.

Looking at the Marvel graph specifically, because there are more movies in the MCU more appeared in the sample. The samples were chosen based off of 80% of the data going into the training environment and 20% going into the testing environment. Red represents the projected worldwide gross and blue represents the actual worldwide gross. In this graph you can see that out of all of the samples, only Iron Man 3's projected gross surpassed the actual.

Looking at the DC graph specifically, the sample is much smaller. Black represents actual worldwide gross and purple represents predicted worldwide gross. In this graph, Wonder Woman's predicted surpassed the actual gross.

MCU Gross Predictions


DCEU Gross Predictions


The second machine learning visualizations we created predict domestic gross based off of international gross and worldwide gross.

Based on the Marvel data, our predictor predicts lower than the actual gross for the most part. Based on this data, Marvel will continue to grow in their total gross domestically, internatonally, and worldwide.

Based on the DC data, our predictor was unable to determine an accurate projection for future growth. More data is needed in order to be able to make an accurate prediction.
